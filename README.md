# rohit-collections

## Overview
This is a public **ReactJS** application built using **PrimeReact UI components** and **SCSS styling**.  
The application fetches data from a **NodeJS backend server** and renders content dynamically based on the currently visited page.

The project focuses on modular architecture, clean UI composition, and scalable data integration.

---

## Tech Stack

### Frontend
- ReactJS
- PrimeReact
- PrimeIcons
- SCSS
- JavaScript (ES6+)
- Axios / Fetch API

### Backend
- NodeJS
- RESTful APIs

---

## Features
- Component-based UI using PrimeReact
- Dynamic page-level data fetching
- SCSS-based styling with modular structure
- Reusable service layer for API communication
- Responsive and modern UI design

---

## Prerequisites
Make sure you have the following installed:

- Node.js (v16 or higher recommended)
- npm or yarn
